# Delivery Pay Tracker (Local-Only)

이 앱은 별도의 서버 없이 브라우저의 `localStorage`에 데이터를 저장하는 개인용 정산 트래커입니다.

## 주요 기능

- **로컬 저장**: 입력한 모든 데이터는 사용자의 브라우저에 실시간으로 저장됩니다.
- **백업 (내보내기/불러오기)**: 상단 헤더의 버튼을 통해 데이터를 JSON 파일로 백업하거나 다시 불러올 수 있습니다.
- **완전 무료 & 프라이버시**: 서버로 데이터를 전송하지 않아 안전합니다.

## 실행 방법

1. 의존성 설치:
   `npm install`
2. 로컬 서버 실행:
   `npm run dev`
3. 버셀 배포:
   이 레포지토리를 버셀에 연결하면 바로 사용 가능합니다 (무료 티어 권장).

## 주의사항

- 브라우저의 캐시나 쿠키를 모두 삭제할 경우 데이터가 사라질 수 있으므로, 중요한 데이터는 정기적으로 **내보내기** 기능을 통해 백업해 주세요.
